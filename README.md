# Final Project - TicTacToe Algorithm

## 🚀 Overview
This project is the culmination of a three-year intensive study (Grades 10-12) in the **Computer Science & Software Engineering** major. It represents a full software development lifecycle, evolving from foundational programming concepts to a complex system integrated with an advanced algorithm.

The project was designed, architected, and implemented as part of my final matriculation (Bagrut) requirements, demonstrating high-level problem-solving and technical proficiency.

---

## 💡 Key Features
* **Three-Year Evolution:** A long-term project that grew in scale and complexity alongside my academic progress.
* **Advanced Algorithmic Core:** Features a sophisticated algorithm optimized for a friendly tictactoe game.
* **Object-Oriented Design (OOP):** Built with a strong emphasis on modularity, inheritance, polymorphism, and clean code principles.
* **Robust Architecture:** Implementation of design patterns and efficient data structures to ensure system stability.

---

## 🛠 Tech Stack
* **Language:** JavaScript
* **IDE/Tools:** Visual Studio
* **Concepts:** Advanced Algorithms, Data Structures, GUI Design, and File I/O.
